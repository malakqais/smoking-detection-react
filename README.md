# SmokeDet — Intelligent Smoking Detection System

A premium React-based dashboard for real-time monitoring and management of smoking violations.

## Features
- **Real-time Dashboard**: Live simulated camera feeds with detection overlays.
- **Advanced Analytics**: Detailed charts and heatmaps using Chart.js.
- **Secure Authentication**: Multi-panel Login/SignUp with password strength analysis.
- **Admin Panel**: Role-based access control and user management.
- **Profile Management**: Customizable profiles with avatar generation.
- **System Settings**: Multi-panel configuration for cameras and detection models.
- **Theming**: Integrated Dark/Light mode persistence.

## Tech Stack
- **Frontend**: React, Vite, React Router, Chart.js, Font Awesome.
- **Backend**: Flask (Python), SQLite.
- **Design**: Premium CSS with Glassmorphism and animations.

## Getting Started
1. Run the backend: `python backend/app.py` (Port 5000)
2. Run the frontend: `npm run dev` (Port 5173)
